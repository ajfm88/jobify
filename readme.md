# Jobify - Full Stack MERN Job Tracking App

## Live Demo:

- [Jobify](https://jobify-ajfm88.onrender.com)

(If it has been a while since the last time the app demo was accessed,
Render will take 45 seconds to a minute to fully load the page.)

## Features:

- 🗄️ MongoDB for database and storage.
- 🛠️ Express.js for building a robust backend and API.
- ⚛️ React.js for a robust front-end.
- 🚀 Node.js for server-side JavaScript execution.
- 💅 Styled with Styled Components.
- 👥 User Profile creation or use the test demo account.
- 🔐 For testing: Click on “Login/Register” and then on “Explore The App.”
