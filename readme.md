# Jobify - Full Stack MERN (MongoDB, Express, React, NodeJS) Job Tracking App

## Live Demo:

- [Jobify](https://jobify-ajfm88.onrender.com)

## Features:

- Uses MongoDB for database and storage. 🗄️
- Styled with Styled Components. 💅
- Front-end built in React. ⚛️
- User Profile creation or use the test demo account. 👥
